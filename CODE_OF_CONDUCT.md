# Code of Conduct

**System Classification**: A.3 He-II λ-Transition κ Analysis  
**Author**: Oleksii Onasenko  
**Developer**: SubstanceNet

## Standards

This project maintains professional academic standards. All participants are expected to:

### Scientific Conduct
- Maintain scientific integrity
- Provide accurate information
- Cite sources appropriately
- Acknowledge limitations clearly

### Professional Behavior
- Communicate respectfully
- Provide constructive feedback
- Focus on technical merit
- Respect diverse perspectives

### Collaboration
- Share knowledge openly
- Help others understand concepts
- Work toward project goals
- Maintain productive discussions

## Unacceptable Behavior

The following are not acceptable:
- Plagiarism or misrepresentation
- Personal attacks or harassment
- Disruptive or unconstructive criticism
- Violation of scientific ethics

## Reporting

Concerns about conduct should be reported by opening an issue on the repository.

## Enforcement

Violations of this code may result in:
- Warning
- Temporary restriction from contribution
- Permanent ban from project participation

Decisions will be made based on severity and context.

## Attribution

This code of conduct applies to all project spaces including repository, issues, and pull requests.
